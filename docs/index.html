Rust Beats is an advanced procedural metal music generation system that creates authentic heavy metal songs using Karplus-Strong synthesis, advanced DSP, and music theory. Each day, a brand new metal track is automatically generated with physically playable riffs, humanized drums, and authentic distorted tones!

‚úÖ 100% Free to Use!

Every song features: 5 metal subgenres (Heavy Metal, Thrash, Death Metal, Doom, Progressive), physically playable guitar riffs generated with fretboard pathfinding, humanized drum programming with velocity randomization and micro-timing, authentic metal tones via Karplus-Strong synthesis, tube distortion, and cabinet simulation, advanced music theory (Phrygian, Locrian, Double Harmonic Major scales), and complex rhythms (Euclidean patterns, polymeters, breakdowns). No samples - everything is synthesized from scratch!
